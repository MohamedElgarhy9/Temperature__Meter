# Temperature__Meter
  ## About the project:
      Temperature Meter supports negative temperature values, measuring and displaying a room temperature on 
      a LCD using: Atmega32A uc, LM35, the internal ADC.
  ### Major components:
   #### External:
      - ATmega32A microcontroller 
      - 2*16 Lcd
      - LM35
   #### Internal:
      - DIO
      - internal ADC
